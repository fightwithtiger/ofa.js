# XDFrame

No webpack,No nodejs,Let the front end go back to the purest web;

XDFrame，让前端回到最纯粹的web；没有nodejs，没有webpack，也能开发超大型应用；

## 为什么叫XDFrame

因为是使用 [Xhear](https://github.com/kirakiray/Xhear) 和 [drill.js](https://github.com/kirakiray/drill.js) 

Xhear 负责视图和数据绑定；

drill.js 负责模块和资源管理；

组合起来能做超大型应用；

两个库加起来也就 45kb 左右；（压缩混淆后的大小）
